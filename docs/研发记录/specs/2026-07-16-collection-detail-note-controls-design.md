# 笔记本与标签详情笔记控件设计

## 目标

让笔记本详情页和标签详情页中的笔记，具备与资料库主页一致的列表/网格视图切换、多选，以及移动到、添加标签、删除批量操作。

## 范围

- 影响笔记本详情页与标签详情页中的笔记内容区。
- 视图模式和多选状态仅属于当前详情页，不与资料库主页或另一详情页共享。
- 沿用既有笔记卡片、圆形封面选择控件、操作菜单和资料库 Repository 操作。

## 方案

- 新增一个资料库 feature 内的共享内容组件，负责渲染创建入口、列表/网格、视图切换、多选与批量操作栏。
- 组件通过回调接收打开、重命名、移动、设置标签、删除笔记等业务操作，不直接访问 Provider 或数据库。
- 笔记本和标签详情页传入各自过滤后的笔记列表和现有 `libraryIndexProvider` 操作回调；组件内部维护瞬态 UI 状态。

## 验证

- 为共享组件增加 Widget 测试，覆盖视图切换、多选进入/退出和批量操作回调。
- 运行相关测试、`flutter analyze` 与全量 `flutter test`。
- 共享 Dart 代码不引入平台判断，所有端走同一 Flutter 渲染路径。
