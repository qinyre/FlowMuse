# 网格封面多选控件设计

## 目标

将笔记、笔记本和标签页面的网格视图多选控件改为圆形，并准确覆盖在封面预览图右上角。

## 范围

- 仅影响资料库、笔记本、标签的网格封面卡片。
- 仅在多选模式下显示。
- 列表视图和标签选择等弹窗中的复选框保持现状。

## 方案

- 在共享组件目录增加一个封面多选控件，统一圆形外观、选中态与触控语义。
- 将控件嵌入每类卡片的封面尺寸区域，使用封面自身的右上角作为定位基准；不再相对整张网格单元定位。
- 沿用现有选择回调和选中状态，不改动资料库、笔记本或标签的状态管理。

## 验证

- 补充或更新相关 Widget 测试，确认网格多选模式中三类卡片使用该控件，列表视图不受影响。
- 运行相关 Flutter 测试、`flutter analyze` 与全量 `flutter test`。
- 检查共享 Dart 代码未引入平台判断，六端沿用同一渲染路径。
