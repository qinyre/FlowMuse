**四、遇到的问题与困难**

- **技术问题：**
  - 白板触控输入存在绘制与画布滑动冲突，且不同设备的手势表现存在差异。
  - **已尝试的解决方案：** 增加手指绘制开关，明确单指绘制、双指滑动的输入规则，并合并至主分支。
  - **需要帮助或待解决：** 无。
  - 鸿蒙 Pen Kit 全局取色与其他平台画布取色的能力、初始位置和坐标表现不同。
  - **已尝试的解决方案：** 将平台差异收敛在对应适配逻辑中，分别修正鸿蒙初始位置与其他端画布取色定位。
  - **需要帮助或待解决：** 无。
  - GitLab CI 在鸿蒙构建环境中未自动识别 Flutter 与 Node 工具链。
  - **已尝试的解决方案：** 显式配置鸿蒙 Flutter 和 Node 路径，并将静态分析调整为仅对 error 阻断流水线。
  - **需要帮助或待解决：** 无。
- **非技术问题：**
  - 结项阶段的功能记录、AI 使用材料和项目文档分散在多个目录，整理成本较高。
  - **已采取的措施：** 统一文档目录职责，整理 AI 使用日志并在协作规范中明确后续文档存放位置。
